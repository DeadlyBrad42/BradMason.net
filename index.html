    A game made in a single weekend!

    December 2012

  • Student's Friend

    A course management system for instructors and students

    April 2012

    Student's Friend is a course management system built by myself and teammates for our Senior Project class at PSU. It offered students a way to manage their coursework, schedule, and get help with assignments. It featured logging in via the Facebook API, some light social integration, and varying levels of access for instructors and students.

    Some of the other features included in the system were a forum for interacting with the instructor and classmates, a flashcard system for study assitance, a seperate calendar for each course, and support for uploading files to share with the class. Instructors have full control of all their classes, and students were able to create flashcards to share with the class, and upload their notes and other files to share with the class.

    The project was all-inclusive from designing the system, implementing it, and documenting all of the features. The system itself was written in PHP, and used JavaScript and jQuery for AJAX and stylized effects. The system ran on an Apache webserver and utilized a MySQL database.

  • Pixel Pipeline

    A prototype raytracing program written in C++.

    ~2007

    Pixel Pipeline was a 3D scene renderer I built in C++ back in 2007. It was able to render scenes made of cubes represented by a three dimensional array. I never quite got the rendering code perfect, but making bitmap images in code always felt pretty awesome.

    Although I lost interest in the project a long time ago, I may give it another shot eventually because the math behind it is pretty cool.
